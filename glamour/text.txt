TDD  =>  Test Driven Development
Разработка основанная на тестах
---------------------------------
RULES (Правила):
1. Write a test (It will fail for 100%)
2. Write the code itself
3. Fix the test
---------------------------------
e2e  -> end to end testing (Robot will use browser and test each and every thing in the project)
unit testing -> each unit of the website (small pieces)
